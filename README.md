# Otvoreni skup podataka o Dota2 turnirima. 

Podacima se modelira pojedinaƒçni turnir te sponzori i njihov odnos koji odgovara many-to-many relaciji. Za te svrhe se u bazi podataka stvara spojna tablica `turnir_sponzor`.
